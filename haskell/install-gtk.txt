[ Installation ]

[ 1]. Install prerequisites
  - sudo apt-get install libglib2.0-dev libcairo2-dev libpango1.0-dev libgtk2.0-dev
  - cabal install cabal-install-1.16.0.2

[ 2]. Install Gtk2Hs itself and some other packages
  - cabal install gtk2hs-buildtools
  - cabal install glib
  - cabal install gio
  - cabal install cairo
  - cabal install pango
  - cabal install gtk

[ 3]. Install additional packages (if necessary)
  - cabal install gconf
  - cabal install glade
  - cabal install gstreamer
  - cabal install gtkglext
  - cabal install gtksourceview2
  - cabal install soegtk
  - cabal install svgcairo
  - cabal install vte
  - cabal install webkit

[ 3]. Install latest cabal-install
  - cabal install cabal-install
